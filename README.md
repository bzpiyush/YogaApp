## Installation

- For Backend

  ```javascript
  npm install
  ```

- For Frontend

  ```javascript
  cd client
  npm install
  ```

## To Run

```javascript
npm dev
```

## Tech Stack Used

- MongoDB
- ExpressJS
- ReactJS
- NodeJS

## Backend Dependencies

- Cors
- Dotenv
- Mongoose
- Validator

## Frontend Dependencies

- Node Sass
- React Router Dom
- React Select
